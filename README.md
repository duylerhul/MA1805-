[01.Drawing](01.Drawing/index.html) 
The motivation for this project originated from the frequent, annoying use of a specific emoji by a friend. My hardest technical challenge, however, was drawing the unique shapes for the feet and hands using only basic p5.js commands. I also struggled to remove the solid black fill inside the eyes and mouth. Later, I discovered the simple solution was removing the CHORD parameter from the arc() functions, which draws an open curve instead of a filled shape. Although I knew the correct fix, I decided to keep the filled shapes because I feel they give the emoji a more intimidating appearance.
